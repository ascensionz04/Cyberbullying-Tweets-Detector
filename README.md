# Cyberbullying-Tweets-Detector
This notebook involves 3 models: RoBERTa, Logistic Regression and Linear SVM and test their ability in detecting toxic comments.

The exported_models will be automatically downloaded after running the Cyberbullying_Tweets_Detector.ipynb (recommended using Google Colab). 

To use the gradio_app, make sure to put the exported_models folder in the same folder as the app.
